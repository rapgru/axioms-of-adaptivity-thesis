# Demonstration Package Documentation

Source code of my bachelors thesis [Formalizing Optimal Convergence of Adaptive FEM in Lean](https://rapgru.github.io/axioms-of-adaptivity-thesis/), documenting the Lean
formalization [axioms-of-adaptivity](https://github.com/rapgru/axioms-of-adaptivity).

## Build Instructions

1. Setup Lean in this project directory.
2. Check out [axioms-of-adaptivity](https://github.com/rapgru/axioms-of-adaptivity) into the folder `../axioms-of-adaptivity`.
3. Run `./build_and_serve.sh` to build and serve the project at `localhost:8000`.
